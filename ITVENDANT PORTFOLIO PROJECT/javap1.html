My Work

  • A Novel Method For Optical Disease Diagnosing Using Resnet 50 Algorithm

    Fundus analysis studies the relationship between human health and changes in the anatomy of the iris. Apart from the fact that Fundus recognition focuses on modeling the overall structure of the iris, Fundus diagnosis emphasizes the detecting and analyzing of local variations in the characteristics of irises. This system presents a computerized iridology system for detecting abnormality conditions in the body. Which is designed with help of image processing through several stages such as pre-processing, feature extraction, classification using threshold Algorithm and blockchain. This system produces accuracy rate of 99.4% which is relatively higher than the existing non invasive models
